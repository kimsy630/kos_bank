Highlight Text

Use the mark element to 형광펜효과 text.

Abbreviations

The abbr element is used to mark up an abbreviation or acronym:

The WHO was founded in 1948.

Description Lists- dl/dt/dd효과

The dl element indicates a description list:

Coffee
- black hot drink
타이틀
- 설명구문이 삽입됨.

Keyboard Inputs

문자열을 키보드와 같이 표현함. 단축키 표현에 유용함

Use ctrl + p to open the Print dialog box.

pre태그 적용하기

pre태그는 text를 편집한 그대로 브라우저에 출력된다. 

    [세종=뉴시스] 임재희 정성
    원 기자 = 18일 0시
       기준 신종 코로나바이
       러스 감염증           (코로나19) 신규 확
       진 환자가 8월29일 이후
        3개월 가
       까운 81일 만에 300명대로 집
       계됐다.
